## Overview

iNet Network Scanner reveals every device connected to your network, providing visibility into home or office infrastructure that router interfaces often obscure. Understanding what's connected represents the foundation of network security and troubleshooting.

Device discovery scans network ranges to identify all connected devices. Each device displays IP address, hostname, and manufacturer identification when available. The discovery reveals devices you may not know exist on your network.

Port scanning checks for open network ports indicating available services. Identify web servers, file sharing, remote access, and other services running on network devices.

Bonjour browser discovers advertised network services for easy access. Find printers, shared folders, and other resources without knowing their addresses.
